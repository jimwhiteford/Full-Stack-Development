# MERN Blog

COMP-2006 | Full-Stack Development

## Description

This is a MERN web application of a blog. It has a home page with minimised cards of all blog posts and a categorys section showing only the relevant blogs. Every blog can be clicked on and taken to the full blog page.

## Getting Started

### Client Dependencies

* Font Awesome
* Jest 
* Jodit Editor
* Socket.io

### Server Dependencies

* Nodemon
* Express
* MongoDB
* Mongoose
* Cors
* cross-env
* dotenv
* socket.io
* Chai
* Jest
* Mocha
* Super Test

### Run Server

* /root
* npm run start

### Run Client

* cd /client
* npm install
* npm run start
